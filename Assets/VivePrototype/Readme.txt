This is a prototype for using the Vive + SteamVR to interact with browsers. Use it as reference code or polish it up for use in your project!

BrowserCursor.cs/BrowserInput.cs:
	Replace these files with the versions that came with ZFBrowser

VRPointerUI.cs:
	An implementation of IBrowserUI for VR. Attach it to a BrowserQuad.

CursorDecal.mat:
	Put this in a Resources/ folder so it can be loaded at runtime.
